# VITAL EDGE

## Overview
VITAL EDGE is a simple medical application designed for our lab assessment at the university. The application provides users with predetermined answers to specific medical questions, facilitating quick access to basic healthcare information.

## Introduction
The purpose of VITAL EDGE is to offer a straightforward platform for users to obtain quick answers to common medical questions. This project aims to enhance understanding of telemedicine applications and their role in providing accessible healthcare information.
(Note: This doesnt use a large dataset to provide answers. The answers and quesries are predetermined in the python code)

## Features
- **Predefined Questions**: Users can select from a list of common medical questions to receive immediate answers.
- **User -Friendly Interface**: The application is designed to be intuitive and easy to navigate.
- **Basic Medical Guidance**: Provides general information and guidance based on the selected questions.

## Installation and Usage
-To set up and use the VITAL EDGE application on your local machine, follow these steps: Clone the repository using `git clone https://github.com/prajwal816/VitalEdge.git`. 
-Navigate to the project directory with `cd VitalEdge`. Install any necessary dependencies (if applicable) using `npm install`. Launch the application with `npm start`.
-Open your web browser and go to `http://localhost:3000` (or the appropriate URL). 
-Enter a question from the list to receive an answer.
